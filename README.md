# functionaly-tg-bot 🤖

Telegram-бот на Python для быстрых действий, генерации случайных значений и обработки изображений.

## Возможности

- 🪙 Подбрасывает монетку.
- 🎲 Бросает кубик.
- 🔐 Генерирует пароль нужной длины.
- 🎰 Выдаёт случайное число в диапазоне.
- 🖼️ Переводит фото в чёрно-белый режим.
- 📌 Управляется через команды и inline-кнопки.

## Команды

- `/start` — главное меню бота.
- `/coin` — подбросить монетку.
- `/dice` — бросить кубик.
- `/password` — сгенерировать пароль длиной 12 символов.
- `/password 20` — сгенерировать пароль длиной 20 символов.
- `/random` — случайное число от 1 до 100.
- `/random 50` — случайное число от 1 до 50.
- `/random 50 100` — случайное число от 50 до 100.
- отправь фото и бот вернёт его в чёрно-белом виде.

## Как работает меню

После команды `/start` бот показывает inline-кнопки:

- Монетка
- Кубик
- Пароль (12)
- Пароль (20)
- Число 1–100

Это позволяет быстро запускать основные функции без ручного ввода команд.

## Установка

### 1. Клонируй репозиторий

```bash
git clone https://github.com/touchfalls/functionaly-tg-bot.git
cd functionaly-tg-bot
```

### 2. Создай виртуальное окружение

```bash
python -m venv venv
```

### 3. Активируй его

**Windows:**
```bash
venv\Scripts\activate
```

**Linux / macOS:**
```bash
source venv/bin/activate
```

### 4. Установи зависимости

```bash
pip install pyTelegramBotAPI Pillow
```

## Настройка

Создай файл `config.py` и добавь в него токен бота:

```python
BOT_TOKEN = "YOUR_TELEGRAM_BOT_TOKEN"
```

## Используемые библиотеки

- `telebot` / `pyTelegramBotAPI` — работа с Telegram Bot API.
- `random` и `string` — генерация случайных значений.
- `Pillow` — обработка изображений.

## Примечания

- Для форматирования сообщений используется `parse_mode="Markdown"`.
- При работе с Markdown в Telegram нужно аккуратно экранировать специальные символы.
- Некоторые импорты могут быть заделом под будущие функции.
